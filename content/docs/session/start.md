<!-- LTeX: language=fr -->
**Lorsque vous commencez un scénario**, effectuez les opérations suivantes:

<!-- LTeX: enabled=false -->
- Identifiez ou ajustez le contenu sensible et [marquez-le comme sensible]({{< ref "docs/session/sensitive" >}}).
<!-- LTeX: enabled=true -->
- Revoyez ou récapitulez les événements du dernier scénario.
- Plantez le décor en visualisant la situation et les intentions
  actuelles de votre personnage.

Vous pouvez également mettre en lumière un nouveau danger, une nouvelle
opportunité ou une nouvelle perspective. Cela peut inclure une scène
cachée à votre personnage. Si vous le faites, visualisez une brève
vignette (vous pouvez lancer un dé ou choisir dans le tableau ci-dessous
pour vous inspirer). Ensuite, tous les joueurs gagnent *+1 momentum*.

<!-- LTeX: enabled=false -->
| Valeurs  | Résultat     |
| :---:    | :--- |
|  1 - 10  | Un flashback révèle un aspect de votre passé ou de votre nature  |
| 11 - 20  | Un flashback révèle un aspect d’un autre personnage, lieu ou faction  |
| 21 - 30  | Un personnage ou une faction influent(e) est présenté(e) ou reçoit de nouveaux détails  |
| 31 - 40  | Des situations apparemment sans rapport se révèlent être liées  |
| 41 - 50  | Des facteurs externes créent un nouveau danger, une nouvelle urgence ou une nouvelle importance pour une quête  |
| 51 - 60  | Un personnage important est mis en danger ou subit une mésaventure  |
| 61 - 70  | Un lieu clé est rendu dangereux ou s’enlise dans un conflit  |
| 71 - 80  | Retour inattendu d’un ennemi ou d’une menace  |
| 81 - 90  | Un péril vous guette ou se cache juste hors de vue  |
| 91 - 100 | Une aide imprévue est en route ou à portée de main  |
