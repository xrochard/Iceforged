<!-- LTeX: language=fr -->
**Lorsque vous terminez un scénario,** repensez aux événements du jeu 
et identifiez les occasions manquées de marquer votre progression.

<!-- LTeX: enabled=false -->
- Si vous avez renforcé vos liens avec un contact, 
[développez cette relation]({{< ref "docs/relationship/development" >}}).
- Si vous avez avancé dans une quête, vous [franchissez une étape importante]({{< ref "docs/quest/reach_milestone" >}}).
<!-- LTeX: enabled=true -->

S’il y a une quête, une relation ou une autre situation sur laquelle vous souhaitez vous concentrer 
lors du prochain scénario, notez-la et gagnez +1 momentum.
